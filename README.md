# blog

springboot博客框架

参考https://github.com/wchstrife/blog写的博客框架

主要使用了springboot+thymeleaf+bootstrap，集成了editormd的markdown编辑器

关于使用，只需要在数据库建一个叫blog的数据库，然后将数据库连接地址改成自己的就ok了

至于在你电脑上会出现什么bug，我也不知，自行解决
